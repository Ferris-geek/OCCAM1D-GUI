# OCCAM1D-GUI

## Pre work

### install packages

```shell
activate MTpy
pip install -r requirements
```

### change errors in MTpy

Go to the occam1d.py, change the 1849-1851 line:`nonposy` -> `nonpostive`

cd OCCAM1D-GUI/code
